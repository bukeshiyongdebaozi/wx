## 如何构建

1. 下载并安装 **Android Studio**。
2. 选择 **Open (打开)** 并选择解压后的文件夹。
3. 等待 Gradle Sync 完成。
4. 点击 **Run (绿色播放按钮)** 以构建并安装到您的设备或模拟器上。
