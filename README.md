由 Web2Native AI 生成。

## 📱 我只有手机，如何获取 APK？ (云端构建)

本项目已配置 GitHub Actions。
1. 将此 ZIP 文件解压。
2. 将文件上传到您的 GitHub 仓库 (Repository)。
3. 点击 GitHub 上的 "Actions" 标签。
4. 等待构建完成，下载生成的 Artifact (app-debug.apk)。

## 💻 电脑构建方法

1. 下载并安装 **Android Studio**。
2. 选择 **Open (打开)** 并选择解压后的文件夹。
3. 等待 Gradle Sync 完成。
4. 点击 **Run (绿色播放按钮)** 以构建并安装到您的设备或模拟器上。
